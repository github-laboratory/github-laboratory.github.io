C言語でHello World

C言語では、printf関数を使用することでコンソールに文字列を出力することができます。 コンソールに出力するためのライブラリが、stdio.hという名前のヘッダファイルです。 stdio.hは、Standard Input Outputのことで、標準入出力という意味です。 これを#includeすることで、標準の入出力先であるコンソール上に入出力が可能になります。

それでは、printf関数を用いてHello Worldの文字列をコンソールに出力してみましょう。 以下のソースコードを見てください。 

#include <stdio.h>

int main(void)
{
printf("Hello World\n");
return 0;
}

まず、#include <stdio.h>でprintf関数を使用可能にしています。 C言語で行いたい処理は、main関数の中に記述します。 main関数は、int型の戻り値を持ちます。 引数のvoidは何もないという意味で、つまりは引数を必要としないということです。 printfは()の中を出力します。 文字列は""で囲みます。 \nは改行を表します。 return 0;でプログラムが正常に終了したことを表します。
