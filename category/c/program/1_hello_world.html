C言語とは


C言語は、1972年にベル研究所から発表されたプログラミング言語です。

数あるプログラミング言語の中でも最初期に開発された言語の一つであり、後発の言語に多大な影響を与えました。

C言語を源流とする、いわゆるC系言語は世界中で幅広い用途に使用され、今でも根強い人気を誇る言語です。

Hello Worldを表示してみよう


さて、開発環境が整ったところで、まずはHello Worldという文字列を表示してみましょう。

これから学ぶプログラムは、Hello Worldと標準出力に表示するプログラムで、プログラミングのチュートリアルとしてよく用いられるものです。

下記のプログラムをテキストエディタで入力し、hello_world.cとして保存してください。

hello_world.c 
//Output String "Hello World"

#include <stdio.h>

int main(void)
{
	printf("Hello World\n");
	return 0;
}

ソースコードの説明


まず、

#include <stdio.h>

の部分について説明します。


#includeは、プリプロセッサ命令と呼ばれる命令の一つで、<>で囲んだヘッダファイルの内容をソースコード中に追加記述するように指示しています。

ヘッダファイル"studio.h"は標準入出力と呼ばれるライブラリで、入出力のカテゴリに属する関数をまとめた辞書のようなものです。

これから使用するprintf関数は、stdio.hの中に含まれています。

つまり、#includeは今からprintfとかいろんな関数を書くけど、関数の詳しい説明は<t;>の中のファイルに書いてあるから参照してね、ということを指示しているわけです。


次に、

int main(void) {}

の部分についてです。


C言語では、通常行いたい処理はmain関数の中に記述します。

これはC言語では特に指示がない場合、main関数をデフォルトの開始位置とする、と定められているためです。

このようなデフォルトの開始位置は各言語で定められており、これをエントリポイントと呼びます。

つまり、C言語のエントリポイントはmain関数です。

さて、main関数の前に記述されているintの部分は戻り値と呼ばれ、intは整数型の変数を表します。

後で詳しく説明しますが、ここではmain関数が整数型の戻り値を持つことを表している、とだけ知っておいてください。

ちなみに、main関数は必ずしもint型の戻り値を持つわけではなく、voidの場合もあります。

()の中身は引数と呼ばれ、関数はこれを受け取って処理に使用します。

処理とは具体的には計算のことです。

引数とは計算を行う材料のようなもので、例えば3+4の計算を行う場合、引数は3と4ということになります。

今回は、main関数にはvoidという引数が与えられています。

voidはその関数に引数がないことを明示的に表し、省略することも可能です。

少し分かりにくいですが、つまり、引数はないけど、()の中が空っぽだと引数が必要なのか、まだ記述していないだけなのか迷ってしまうかもしれないから、 分かりやすいようにvoidと書いておく、ということです。

プログラミングも大規模なものになると、とても一人では手に負えず、チームで仕事をすることになります。

自分だけではなくて、ソースコードをチェックする仲間にも分かりやすく。

これがプログラミングの基本です。


さて、main関数の中身を見ていきましょう。

printf("Hello World\n");


printf関数は、()の中身を出力します。

文字列を出力する場合は""で囲みます。

\nはエスケープシーケンスの一つで、改行を表します。

省略してもエラーにはなりません。

しかし、実行してみると分かりますが、省略すると改行が行われず綺麗に表示されません。

また、C言語では関数の最後に必ず;(セミコロン)を付けなければいけません。

return 0;


return文で、関数の戻り値を設定することができます。

return 0;は、main関数の戻り値を0に設定しています。

0は通常、正常終了を表し、システムにプログラムが正常終了したことを伝えます。

以上が、Hello Worldを標準出力に出力するプログラムです。

次回から、C言語の基礎を詳しく解説していきます。
