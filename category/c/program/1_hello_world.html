Hello Worldを表示してみよう


C言語に限らず、プログラミング学習の最初のステップはHello Wolrdという文字列を表示するプログラムを作成することであるとされています。

"Hello World"という文字列自体に特に意味はありません。

別に"こんにちわ"でも、"お腹空いた"でも構わないのですが、何事もまずは挨拶をしっかりとしておきましょう、ということのようです。

最も基本的なプログラムであり、言語の特徴を把握するにはぴったりです。

C言語では、printf関数を使用することでコマンドラインに文字列を出力することができます。

コマンドラインとは、映画などでハッカーがカタカタやっている、あの黒い画面のことです。

ひとまず、ソースコード(プログラムの設計図)を見てみましょう。

hello_world.c 
								
#include <stdio.h>

int main(void)
{
	printf("Hello World\n");
	return 0;
}

ソースコードの説明


まず、

#include <stdio.h>

について説明します。


#includeの部分は、<>で囲んだヘッダファイルをソースコードに追加するように指示しています。

ヘッダファイル"studio.h"は標準入出力と呼ばれるライブラリで、入出力のカテゴリに属する関数をまとめた辞書のようなものです。

printf関数は、stdio.hの中に含まれています。


次に、

int main(void) {}

の部分についてです。


C言語では、通常行いたい処理はmain関数の中に記述します。

intの部分は戻り値と呼ばれ、ここではmain関数が整数型の戻り値を持つことを表しています。

戻り値は、関数が結果的に出力する値のことで、他の関数で受け取って使用します。

()の中身は引数と呼ばれ、関数はこれを受け取って処理に使用します。

voidは、この関数は引数を必要を必要としませんよ、ということを表します。


さて、main関数の中身を見ていきましょう。

printf関数は、()の中身を出力します。

文字列を出力する場合は""で囲みます。

\nは改行を表し、無くてもエラーにはなりませんが、ターミナルにきれいに出力するためには必須です。

関数の最後には必ず;を付けなければいけません。

returnで、戻り値を設定することができます。

return 0;は、main関数の戻り値となり、プログラムが正常に終了したことを伝えます。

今回は値を受け取る関数がありませんが、この一文を追加しておく方が無難です。

0である必要はなく、1や-1を返すことも可能です。

以上をまとめると、このプログラムはstdio.hを使用し、その中に含まれるprintf関数でHello Worldの文字列を出力し、正常終了したことを伝えています。
