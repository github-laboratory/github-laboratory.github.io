Hello Worldを表示してみよう


さて、開発環境が整ったところで、まずはHello Worldという文字列を表示してみましょう。

hello_world.c 
//Output String "Hello World"

#include <stdio.h>

int main(void)
{
	printf("Hello World\n");
	return 0;
}

ソースコードの説明


まず、

#include <stdio.h>

の部分について説明します。


#includeは、プリプロセッサ命令の一つで、<>で囲んだヘッダファイルの内容をソースコード中に追加記述するように指示しています。

ヘッダファイル"studio.h"は標準入出力と呼ばれるライブラリで、入出力のカテゴリに属する関数をまとめた辞書のようなものです。

これから使用するprintf関数は、stdio.hの中に含まれています。


次に、

int main(void) {}

の部分についてです。


C言語では、通常行いたい処理はmain関数の中に記述します。

intの部分は戻り値と呼ばれ、ここではmain関数が整数型の戻り値を持つことを表しています。

()の中身は引数と呼ばれ、関数はこれを受け取って処理に使用します。

voidの場合、その関数は引数を必要としないことを明示的に表し、省略することも可能です。


さて、main関数の中身を見ていきましょう。

printf("Hello World\n");


printf関数は、()の中身を出力します。

文字列を出力する場合は""で囲みます。

\nはエスケープシーケンスの一つで、改行を表します。

省略してもエラーにはなりません。

しかし、実行してみると分かりますが、省略すると改行が行われず綺麗に表示されません。

また、C言語では関数の最後に必ず;(セミコロン)を付けなければいけません。

return 0;


return文で、関数の戻り値を設定することができます。

return 0;は、main関数の戻り値を0に設定しています。

0は通常、正常終了を表し、システムにプログラムが正常終了したことを伝えます。
